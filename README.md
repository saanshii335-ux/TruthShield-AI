🛡️ TruthShield AI

TruthShield AI is an AI-powered web application designed to detect potential misinformation in social media posts. 
It combines Machine Learning, OCR, and Google Gemini AI to analyze text and image-based content, providing users with confidence scores,
AI-generated reasoning, and detection history.

🚀 Features
 1-Text-based fake news detection
 2-Image upload with OCR extraction
 3-Machine Learning prediction
 4-Confidence score with trust bar
 5-Google Gemini AI reasoning report
 6-SQLite detection history
 7-Responsive drag-and-drop interface
 8-Cloud deployment using Render
